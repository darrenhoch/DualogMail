DualogMailTool is a powerful utility designed to help system administrators fix common webmail database issues, manage IMAP servers, and troubleshoot email delivery problems. It provides an easy-to-use menu-driven interface for maintaining and diagnosing Dualog email infrastructure.    

- Fixes corrupted or inconsistent webmail database records
- Restores access to inaccessible email folders
- Provides diagnostic tools for IMAP and SMTP servers
- Helps identify and manage large files and attachments
- Enables system tools for advanced troubleshooting
